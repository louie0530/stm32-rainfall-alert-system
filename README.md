# STM32 Rainfall Alert System

## Overview
This project detects rainfall based on sound intensity using a microphone sensor.
When the sound level exceeds a predefined threshold, an alert is triggered.

## Hardware
- STM32 NUCLEO-F401RE
- Electret Microphone Module
- Buzzer
- LED
